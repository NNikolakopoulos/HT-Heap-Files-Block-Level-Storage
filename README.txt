hashFunStrings:
ειναι μια απλη hash function για strings που χρησιμοποιειται για το hashing των surname του SHT 





SHT_InsertEntry: 

χρησιμοποιουμε την hashFunction για να δουμε σε ποιο bucket πρεπει να εισαχθει το srecord
μολις βρουμε το bucket κοιταμε αν ειναι αδειο

αν ειναι αδειο δημιουργουμε ενα νεο μπλοκ και αποθηκευουμε τα στοιχεια μας

αν δεν ειναι αδειο τοτε βρισκουμε ποσα srecords εχει μεσα το μπλοκ που ανοιξαμε και αν χωραει κιαλο το αποθηκευουμε εκει αλλιως δημιουργουμε ενα νεο μπλοκ
Υπάρχει έλεγχος για διπλότυπα.


SHT_InsertALL:
ανοιγουμε το πρωτευον ΗΤ και διαβαζουμε ενα ενα τα records και εισαγουμε για καθε record,το blockid και το surname  με την SHT_InsertEntry sto SHT 





SHT_SecondaryGetAllEntries:
καταρχας ελεγχουμε αν το value ειναι NULL
αν ειναι τοε απλα καλουμε την HT_GetAllEntries για να τα εκτυπωσει ολα.

σε διαφορετικη περιπτωση ψαχνουμε καθε στοιχειο του SHT και να βρουμε το ιδιο surname τοτε παμε στο μπλοκ με blockid στο HT και κανουμε print τα στοιχεια του SHT
αλλιως εκτυπωνεται μηνυμα λαθους
